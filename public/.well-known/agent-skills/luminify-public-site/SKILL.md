# Luminify Public Site

## When to use

Use this skill when a user wants to understand Luminify, browse public tools, compare pricing, read FAQs, or navigate the live site without logging in.

## Public pages

- `/` landing page with the pricing section and FAQ
- `/about`
- `/faq`
- `/contacts`
- `/tools`
- `/tools/background-remover`
- `/tools/white-background`
- `/tools/blur-background`
- `/tools/change-background-color`
- `/tools/add-background`
- `/tools/photo-enhancer`
- `/privacy`
- `/terms`

## Guidance

- Prefer public marketing pages for product overviews and public tool details.
- Use `/.well-known/api-catalog` for machine-readable API discovery.
- If browser agent features are available, prefer the site’s WebMCP tools for navigation tasks.
